# Projeto TecBoard

Este é o projeto **TecBoard**, uma landing page focada em uma plataforma de monitoramento em tempo real com alertas inteligentes. O projeto foi desenvolvido como parte de um curso de formação em Front-End.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout adaptativo (Desktop, Tablet e Mobile) e uso de `@font-face` para tipografia personalizada.

## 💻 Sobre o Projeto

O TecBoard visa oferecer uma interface limpa e moderna para usuários que precisam gerenciar o uptime de suas aplicações. O foco do desenvolvimento foi:

* **Adaptabilidade:** Adaptação de layout para diferentes resoluções de tela.
* **Tipografia:** Uso de fontes personalizadas (Unbounded e Poppins) para reforçar a identidade visual.
* **UX/UI:** Design focado em conversão, com destaque para a chamada de ação (Call to Action).
